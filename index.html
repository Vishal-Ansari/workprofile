Enterprise HR Data Integration & Identity Provisioning Platform

Azure Databricks · Dayforce HCM · MidPoint IDM · Cornerstone LMS · REST APIs · Unity Catalog

Designed and implemented an automated HR integration platform that replaced a fully manual 3-system handoff with a real-time, fault-tolerant data pipeline. Synchronizes employee lifecycle data — hires, updates, terminations — across Dayforce, MidPoint, and Cornerstone on the same day events occur.

Key Contributions
Medallion Architecture — All 4 layers (Bronze → Silver → Core → Writeback) as SSOT for 5,000+ employee records across 3 systems.
Ingestion Framework — 3-notebook pipeline: XRef discovery, parallel extraction, DLQ-based replay with watermark lineage.
Failure Recovery — Dead Letter Queue with bounded retry cycles, reducing reprocessing effort by 80%.
PII Masking — 20+ PII fields protected via Unity Catalog Governed Tags, ABAC policies, custom masking functions.
Identity Provisioning — Automated Dayforce ↔ MidPoint GID ↔ Cornerstone linking — same-day onboarding and termination.
Zero Manual Touchpoints — 100+ daily events, 3,000+ monthly, zero manual intervention.
